# EduTime - PWA (Web + App)

Este repositorio contiene la versión web completa de **EduTime** diseñada para funcionar como PWA.
Coloca estos archivos en la rama `main` y activa GitHub Pages (raíz) para publicar.

**Archivos incluidos**
- index.html
- styles.css
- app.js
- manifest.json
- service-worker.js
- content.json
- /assets/logo-192.png
- /assets/logo-512.png

**Editar contenido**
Para cambiar títulos, eslogan o retos permanentemente, edita `content.json` en GitHub (desde la app o la web).

**Convertir a APK**
Para convertir a APK usa PWABuilder (https://www.pwabuilder.com) o Web2Apk y pega la URL pública de GitHub Pages.
